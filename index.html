Question 7: Why do you think SSL/HTTPS is important?

SSL/HTTPS is important, because it's more secure. With SSL, sensitive information was encrypted, unreadable to everyone except for the server you are sending the information to. This protects it from hackers and identity thieves.

SSL also provides Trust. A website using SSL/HTTPS with a CA signed, will be reliable, and you can feel comfortable to input your credit card infomation on it.

The latest version of SSL Protocol is 1.2, and I'm sure that a lot of customer will request to apply it, to make sure their website is compliance with any security scan.
